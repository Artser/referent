# Referent - Next.js приложение

Минимальное приложение на Next.js с TypeScript.
PROJECT.MD - описание проекта
## Установка

Установите зависимости:

**Через pnpm (рекомендуется):**
```powershell
pnpm install
```

**Через npm:**
```powershell
npm install
```

## Запуск

Запустите сервер разработки:

**Через pnpm:**
```powershell
pnpm dev
```

**Через npm:**
```powershell
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Скрипты

- `pnpm dev` / `npm run dev` - запуск сервера разработки
- `pnpm build` / `npm run build` - сборка для продакшена
- `pnpm start` / `npm run start` - запуск продакшен сервера
- `pnpm lint` / `npm run lint` - проверка кода линтером
